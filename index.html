1. World Wide Web

Is a collection of HTML documents. These are made of something called HTML, which stands for HyperText Markup Language. This is the basis for almost every webpage. Hyperlinks are what makes the makes the web web-like.

The web was invented in the early 1990s and has something on the order of 30 billion pages.

Anything can exist on the web: plain text, html, music, images, videos, etc.

2. HTML

(HyperText Markup Language)

The heart of the web.

HTML is made up of:

  • text content - what you see
  • markup - what it looks like
  • references to other documents - e.g. images + videos
  • links to other pages

In HTML:

  • plain text is plain text
  • HTML Markup - is made up of things called tags.
  • < b>anything< /b> ----> bold
  • < em>anything< /em> ----> italics

If you don't use a closing tag after the opening tag, everything after the tag will be considered part of it.

3. HTML Attributes

Composition:

  • < tag attr="value">contents< /tag>

Examples of tags that use an attribute:

Anchor tag - < a>

  • < a href="www.reddit.com">Reddit< /a>

Image tag - < img>

  • < img src="url" alt="text"> ----------> src = source
  • < img src="" alt=""> ----------> void tag - closes without tag because there is no content

Whitespace

To force space you need to use the void tag < br> = break = line breaks.You put it after the word you want the line to break at. If not html puts everything in one line. Can also use the < p> tag. Is not a void tag, because it has content.

Content can be either inline or block

  • "br" - inline
  • "p" - block - makes an invisible block (can actually have height and width)

Container elements:

Elements that do nothing but contain stuff inside of them
  • < span> text < / span> ---> inline
  • < div> text < / div> ---> block

with CSS

  • < span class="foo"> text < / span>
  • < div class="bar"> text < / div>

  • Inline Elements so far = a, span, br, img, strong
  • Block Elements so far = div, p, form
